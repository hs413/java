## 생성자에 매개변수가 많다면 빌더를 고려
### 점층적 생성자 패턴
- 매개변수의 개수 별로 생성자를 정의하는 방식
  - 매개변수가 많아지면 클라이언트 코드를 작성하거나 읽기 어렵다

### 자바빈즈 패턴 
- 매개변수가 없는 생성자로 객체를 만든 후 세터 메서드를 호출해 값을 설정하는 방식
  - 객체 하나를 만들기 위해 메서드를 여러 개 호출해야 한다.
  - 객체가 완전히 생성되기 전까지 일관성이 무너진 상태에 놓이게 된다.
  - 클래스를 불변으로 만들 수 없다.
